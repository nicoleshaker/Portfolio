I used: HTML, external CSS, JavaScript

This is a webpage I designed and developed that takes four user inputs to generate a unique, personalized Spotify playlist. I made it in 2021 for a school project with the intention of helping my classmates discover new music. The four user inputs required to produce the playlist are “Artist You Like,” “Song You Like," “Genre You Like,” and “Genre You Want to Get Into.” After the user inputs all the values, they click the "MAKE ME A PLAYLIST" button, after which a unique token is generated. I use this token to produce the link to the playlist, which the user can then listen to and add to their library.
